Programación web

En esta sección encontrarás algunos de mis proyectos relacionados con el desarrollo web, donde aplico lenguajes y tecnologías como HTML, CSS, PHP y Java. Cada trabajo muestra diferentes habilidades en estructura, diseño, funcionalidad y lógica de programación.

HTML y CSS

Desarrollo de interfaces web utilizando HTML para la estructura del contenido y CSS para el diseño visual. Incluye maquetación responsiva, uso de flexbox, grid, estilos personalizados y optimización para diferentes dispositivos.

PHP

Implementación de lógica del lado del servidor mediante PHP. Incluye manejo de formularios, validación de datos, interacción con bases de datos y generación de contenido dinámico para mejorar la funcionalidad del sitio.

Java

Desarrollo de aplicaciones y programas básicos en Java, aplicando principios de programación orientada a objetos, control de flujo, mandar a llamar texto, clases, métodos y estructuras de datos. Ideal para la construcción de herramientas y lógica más avanzada.
