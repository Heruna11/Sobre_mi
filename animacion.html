Modelado 3D

En esta sección encontrarás una recopilación de mis trabajos realizados en modelado 3D, papercraft y más. Cada proyecto representa una parte de mi estilo y aprendizaje en esta disciplina.

Luces

Configuración y estudio de iluminación en 3D, experimentando con luces direccionales, ambientales y de relleno para generar atmósferas realistas. Incluye pruebas de sombras, intensidad, color y rebotes para lograr una composición visual equilibrada.

Materiales

Creación y configuración de materiales utilizando nodos y mapas de textura para lograr superficies realistas. Incluye ajustes de color, rugosidad, metalicidad y normales, así como pruebas de iluminación para asegurar que los materiales reaccionen de forma correcta en diferentes entornos.

Papercraft

Diseño y creación de una figura estilo papercraft, elaborando patrones desplegables en 2D a partir de un modelo 3D. Incluye corte, armado manual y ajuste de pliegues para obtener una estructura final limpia y estable.
